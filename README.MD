# INSTALANDO O PACOTE

1. composer require lsiqueira/laziness

2. Importar a a lib no PRIVIDER(config/app.php)
   
    'providers' => [
        lsiqueira\laziness\LazinessServiceProvider::class,
    ]

3. Publicar os componentes VUE 

    php artisan vendor:

g    
4. Rodar o npm:
    npm install
    npm run dev
